### Download

The Windows versions of **xPack GNU Bash**
are packed as `.zip` files.
Download the latest version named like:

* <code>xpack-bash-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-win32-x64.zip</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.
The `win32` part is confusing, but we have to live with it.

:::

### Unpack

To manually install the <b>xPack GNU Bash</b>,
unpack the archive and move it to a location of your choice.

The recommended location is
the <code>%USERPROFILE%\\AppData\\Roaming\\xPacks\\bash</code> folder, for
example <code>C:\\Users\\ilg\\AppData\\Roaming\\xPacks\\bash\\xpack-bash-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }</code>.

:::note

According to Microsoft, <code>AppData\\Roaming</code> is the recommended location for
installing user specific packages.

:::
