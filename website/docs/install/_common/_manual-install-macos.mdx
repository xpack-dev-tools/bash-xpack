### Download

The macOS versions of **xPack GNU Bash**
are packed as `.tar.gz` archives.
Download the latest version named like:

* <code>xpack-bash-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-darwin-x64.tar.gz</code>
* <code>xpack-bash-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }-darwin-arm64.tar.gz</code>

:::note

In case you wonder where the suffix comes
from, it is exactly the Node.js `process.platform` and `process.arch`.

:::

### Unpack

To manually install the <b>xPack GNU Bash</b>,
unpack the archive and move it to a location of your choice.

The recommended location is
the <code>~/Library/xPacks/bash</code> folder, for
example <code>/Users/ilg/Library/xPacks/bash/xpack-bash-{ customField('upstreamVersion') }-{ customField('xpackSubversion') }</code>:

<CodeBlock language="console"> {
`mkdir -p ~/Library/xPacks/bash
cd ~/Library/xPacks/bash

tar xvf ~/Downloads/xpack-bash-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }-darwin-x64.tar.gz
chmod -R -w xpack-bash-${ customField('upstreamVersion') }-${ customField('xpackSubversion') }`
} </CodeBlock>
